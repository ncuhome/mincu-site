# iNCU 开发者模式

## 加入开发者模式
这一步在安装应用后，数据未清空前，只需要操作一次。

- 首先，下载最新版的[ iNCU](https://incu.ncuos.com/)，然后打开应用 - 切换到生活页 - 右上角扫一扫如下二维码：

![image.png](https://cdn.nlark.com/yuque/0/2022/png/471059/1649151298983-8699fe4a-9a78-465a-ac53-312bc71090ac.png#clientId=ue298bb3e-ee6e-4&crop=0&crop=0&crop=1&crop=1&from=paste&height=250&id=ub96de4c5&margin=%5Bobject%20Object%5D&name=image.png&originHeight=500&originWidth=500&originalType=binary&ratio=1&rotation=0&showTitle=false&size=9671&status=done&style=none&taskId=ua6899449-40f0-45b5-9e35-5fcfd5ddc55&title=&width=250)

- 扫码完毕后，会提示已加入开发者模式，然后 生活页 底部会出现 测试功能板块， 如：

![image.png](https://cdn.nlark.com/yuque/0/2022/png/471059/1649151607252-b0f32342-d816-42c0-bfff-dfd008bef1f8.png?x-oss-process=image/format,png#clientId=ue298bb3e-ee6e-4&crop=0&crop=0&crop=1&crop=1&from=paste&height=1200&id=u6a479f31&margin=%5Bobject%20Object%5D&name=image.png&originHeight=2400&originWidth=1080&originalType=binary&ratio=1&rotation=0&showTitle=false&size=7780774&status=done&style=none&taskId=u10e5b373-a5fd-4f08-8dc6-c2234a7287f&title=&width=540)

## 添加测试页面

点击测试功能右侧的 + 号，添加测试页面：
> 这里添加的页面地址必须是合法的 URL 地址，否则无法添加，需要带协议前缀， HTTP/HTTPS
