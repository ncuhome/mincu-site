---
title: ❓ 什么是 mincu？
---

iNCU 为其内嵌的 WEB 页面提供了丰富的 JS API，用来扩展内嵌页面 JS 的能力，如获取用户信息、操作原生 UI、获取原生 UI 和数据调用能力，以及解决你最头疼的认证问题（登录注册）。

而 mincu 是南大家园小程序生态中最基础的开发框架，它整合了项目初始化模板、移动端容器调试、原生级别 API 调用以及登录信息获取等功能，你想要的这里全都有，而且所有的能力全部开箱即用！

## 🎮 为什么要用？

1. 定制属于你的 webview，做最极致的用户体验
1. 让你的应用在诞生之际就能获得上万级别的用户
1. 无需再为调试客户端头疼，使用 Web 开发技术即可获得和 Native 一样的体验
1. 跨平台调用原生能力
1. 可唤起系统原生的 UI 界面/组件，让 Web App 更 Native 化
1. 拥有白名单机制，大幅度提高安全性与稳定性
1. 使用客户端分享功能，打通用户分享转发链路
